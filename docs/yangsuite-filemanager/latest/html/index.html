yangsuite-filemanager¶

Provides quick, low-overhead parsing of YANG (RFC 6020, RFC 7950 models and identification of their interdependencies. Manages YANG file repositories and sets of YANG files within these repositories. Provides UI and APIs for file upload to YANG Suite.

API is through the class YSYangSet which additionally implements the pyang.Repository interface.
