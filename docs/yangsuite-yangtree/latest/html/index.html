yangsuite-yangtree¶

Core plugin for YANG Suite.

Manages loading, caching, and validation of YANG (RFC 6020, RFC 7950) models. Represents parsed YANG models as Python dicts used in plugins.. Adds GUI for traversing, searching, and inspecting YANG model trees.
