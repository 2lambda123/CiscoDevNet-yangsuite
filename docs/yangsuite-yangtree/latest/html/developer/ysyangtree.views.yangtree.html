ysyangtree.views.yangtree.get_tree(request)[source]¶

Retrieve context and build the JSTree based on module name.

Parameters:request (django.http.HttpRequest) –

HTTP POST request

  • ref (str): reference for finding shared context (default: username)
  • name (str): module name to be parsed (DEPRECATED)
  • names (list): module name(s) to parse
  • yangset (str): YANG set name ‘owner:setname’ to use with context.
  • included_nodetypes (list): nodetypes to include in the tree. If unspecified, defaults to ymodels.DEFAULT_INCLUDED_NODETYPES.
Returns:django.http.JsonResponse – JSTree
