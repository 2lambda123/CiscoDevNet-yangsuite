yangsuite.plugins¶

yangsuite.plugins.call_pip(args)[source]¶

Invoke pip with the given list of args and return output as a string.

yangsuite.plugins.check_for_plugin_updates()[source]¶

Check upstream repository for available YANG Suite plugins.

Since this involves network operations it can be somewhat slow, which is why this is a separate function from get_plugin_versions().

Returns:list – of dicts with keys ‘package_name’, ‘latest_version’, ‘description’, or an empty list in case of various failures.
yangsuite.plugins.get_plugin_versions()[source]¶

Get the installed YANG Suite plugins and their versions.

Returns:list –
of dicts with keys ‘package_name’, ‘installed_version’,
’module_name’, ‘verbose_name’, ‘error_message’

yangsuite.plugins.update_plugins(plugins)[source]¶

Update the given plugin to the latest release or pre-release version.

Parameters:plugins (list) – of dictionaries with keys ‘plugin’, ‘version’
Returns:dict – Package names as keys, each value is one of “updated”, “unchanged”, or “failed”
