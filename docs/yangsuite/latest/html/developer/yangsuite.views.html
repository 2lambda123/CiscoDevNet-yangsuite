yangsuite.views¶

Baseline views provided by yangsuite itself.

yangsuite.views.check_plugins(request)[source]¶

Check upstream repository for new versions of installed plugins.

Since this involves network operations it can be somewhat slow, which is why this should be invoked as an AJAX call rather than as part of the page load in plugins() above.

Returns:django.http.JsonResponse – {‘apps’: [{‘package_name’: ‘yangsuite’, …}]}
